# Product Concept: Life Script (Provisional Name)

## 1. Product Vision
**"Your Life's Algorithm."**
自分の価値観や将来のビジョンから逆算し、人生という壮大なプロジェクトのロードマップを描き出し、それに基づいた「今やるべき行動」を導き出すAIキャリア＆ライフプランナー。

## 2. Target User & Philosophy
- **Target**: 将来の目標（30歳までに起業、海外移住、FIREなど）はあるが、今から何をしていけばいいか逆算できていない人。自分の価値観や強みを言語化し、それに沿った人生設計をしたい人。
- **Philosophy**: 「なんとなく生きる」を脱却する。個人の価値観と将来像をアルゴリズム（道筋）に落とし込み、日々のアクションと結びつける。

## 3. Core Features (面白い機能のアイデア)

### ① Value Extraction (価値観の発掘と明文化)
- 対話型AIによる深いヒアリング機能。「どんな瞬間に幸せを感じるか」「過去の挫折経験」などをAIとチャットする中で、潜在的な価値観や持論を言語化し、「My Philosophy（私の哲学）」としてドキュメント化する。

### ② Life Reverse-Engineering (人生の逆算とロードマップ生成)
- 設定した最終ゴール（例：5年後に独立して年商1億）から、AIが逆算してロードマップを自動生成する。
- 「3年後」「1年後」「半年後」「今月」の具体的なマイルストーンを定義する。
- **面白要素（If-Thenシナリオ）**: 「もしプランA（起業）が失敗した場合のプランB（フリーランスエンジニア）」といった、複数ルートのシミュレーションとリスクヘッジもAIが提示。

### ③ Reality Check & Pivot (現実とのギャップ診断と軌道修正)
- 定期的なレビュー機能（月次・四半期）。「今月予定していたマイルストーンに対して、現状どうですか？」とAIが壁打ち相手となり、進捗が遅れている場合はロードマップの引き直し（ピボット）を提案。
- 外部データ（市場動向、必要なスキルセットのトレンド）をAIが分析し、「最近、AIの進化によりこのスキルの重要度が下がっているので、目標設定をXXに微調整しませんか？」という外部コンテキストを含んだ提案。

### ④ Output Generation (履歴書・職務経歴書の自動生成)
- 日々の進捗や達成したマイルストーンをログとして蓄積し、それをもとにいつでも最新の「魅力的な職務経歴書」や「ポートフォリオ」をAIが自動生成する。

## 4. 独立させるべきか？ (Scriptとの棲み分け)

結論として、**まずは別のアプリ（または明確に分かれた上位モジュール）として独立させる方が良い**です。

### 独立させるべき理由：
1.  **時間軸と利用頻度の違い**:
    *   **Script (TODO/実行支援)**: 時間軸は「今日〜今週」。利用頻度は「毎日・毎時間」。**実行**にフォーカス。
    *   **Life Script (人生設計)**: 時間軸は「年〜数十年」。利用頻度は「月に1回〜数ヶ月に1回」の深い内省。**計画・戦略**にフォーカス。
    これらを同じ画面に混ぜると、日々のタスク管理ツールとしては重苦しくなり、「牛乳を買う」と「3年後の起業準備」が並立して認知負荷が跳ね上がります。

2.  **ユーザーの心理的モードの違い**:
    *   TODOアプリを開く時は「さあ、やろう（DO）」というモード。
    *   人生設計をする時は「立ち止まって考えよう（THINK）」というモード。

3.  **プロダクトの複雑化回避**:
    *   初期（MVP）段階からこれを組み込むと、機能が発散し、本来の「タスクを分解し実行のハードルを下げる」というコアバリューがブレてしまいます。

### 将来的な連携構想 (エコシステム):
独立したアプリとして育てた後、データレベルで連携させるのが最も美しい形です。

*   **Life Script**: 上流工程（戦略）。目標と年次/月次のマイルストーンを決定する。
*   **Script (TODO)**: 下流工程（戦術）。Life Scriptで決まった「今月のマイルストーン」をインポートし、それを日々の実行可能なタスクに**Magic Breakdown**して実行支援する。

まずは、目の前の「実行の摩擦をゼロにする」 **Script (TODOアプリ)** の開発に集中し、そこで得た知見（AIによるタスク分解の精度やユーザー行動データ）を活かして、ゆくゆくは上位概念である人生設計アプリへと拡張或いは連携していく、というアプローチが成功確率が高いと考えます。
